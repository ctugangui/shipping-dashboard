# Task: Fix Docker Start Script

**Context:** The Docker container fails to start because it cannot find `/app/dist/server.js`.
**Reason:** Removing `rootDir` in the previous step caused TypeScript to output the build to `dist/src/server.js` instead of `dist/server.js`.

# Requirement
Update `package.json` to point to the correct build artifact.

* **File:** `package.json`
* **Change:** Update the `start` script.
    * **From:** `"start": "node dist/server.js"`
    * **To:** `"start": "node dist/src/server.js"`

# Verification
1.  Run `docker-compose up --build`.
2.  Success is defined by seeing the log: `Server running on http://0.0.0.0:3000`.
